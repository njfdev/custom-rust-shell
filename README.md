# Custom POSIX-Compliant Shell in Rust

This is a simple POSIX-compliant shell written in Rust.

It supports the following commands:

- `exit`
- `echo`
- `type`
- `pwd`
- `cd`

It also supports running any executable in your $PATH environment.
